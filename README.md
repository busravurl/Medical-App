

# Getting Started


```bash
# using npm
npm start

# OR using Yarn
yarn start
```


### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
